# NavBar

NavBar provides access to destinations and is permanently on-screen.

It can contain:

- Five to seven main destinations (navButtons)
- Five max transversale destinations (navIcons)

Play with NavBar live on [CodeSandbox](https://codesandbox.io/s/sidebar-demo-t4s312)
