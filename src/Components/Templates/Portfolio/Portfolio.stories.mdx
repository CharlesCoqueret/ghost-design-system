# Portfolio

The portfolio is a high order component containing a <LinkTo story='filter'>Filter</LinkTo> controlling a <LinkTo story='static-data-table'>StaticDataTable</LinkTo>.
It can handle:

- pagination with infinite scrolling
- sort of the data
- filtering of the data
  if the provided webservice supports it.
